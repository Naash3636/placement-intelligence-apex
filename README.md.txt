# Placement Intelligence Apex

AI Powered Placement Analytics System built using Streamlit.

## Features

• University placement dashboard  
• Student performance analytics  
• Company recruitment insights  
• AI dataset analyst  
• AI graph generator  
• Placement prediction analytics  

## Technologies Used

Python  
Streamlit  
Pandas  
Plotly  
Scikit-Learn  

## How to Run

pip install -r requirements.txt

streamlit run app.py
